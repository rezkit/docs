RezKit Booking Manager is simple but powerful reservations system where you can view, create and manage your customers bookings.

These bookings could be Bookings on products from RezKit Tour Manager or, RezKit Booking Manager can be integrated with any travel prioduct system.

RezKit Booking Manager has powerful features for managing bookings such as status management, customer messaging, reports, transaction management, tagging, booking activity log, custom passenger fields, multi currency booking and more.
